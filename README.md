# SWIR_Plastic_Sort
- 利用1140/1200/1245/1310nm的多光谱数据，实现对PET、PVC、PP、PC、PE、PS、ABS、SX、PA、PMMA等塑料材质的精细分类
- 4个滤光片参数分别为：
中心波长1140nm,半带宽10nm,峰值透过率90%；
中心波长1200nm,半带宽25nm，峰值透过率80%；
中心波长1245nm,半带宽13nm，峰值透过率90%；
中心波长1310nm,半带宽25nm,峰值透过率80%。
- label的对应关系
    1: "PA",
    2: "PET",
    3: "PMMA",
    4: "PP",
    5: "PS",
    6: "PVC",
    7: "PC"
