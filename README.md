# SWIR_Plastic_Sort
- **代码目录结构**
  - dataset（数据集）：包含高光谱图像(HSI)和多光谱图像(MSI)数据，分别各来自两批塑料样品
  - band_selction(波段选择方法)：从高光谱数据的224个波段（900-1700nm）中筛选出4个特征波段，降维成多光谱数据，以降低系统成本
  - sort（分类方法）：利用1140/1200/1245/1310nm四个波段的多光谱数据，实现对PET、PVC、PP、PC、PE、PS、PA、PMMA等8种塑料材质的精细分类
  - diagram：论文中的图片源文件
- **4个滤光片的参数**
  - 中心波长1140nm,半带宽10nm,峰值透过率90%
  - 中心波长1200nm,半带宽25nm，峰值透过率80%
  - 中心波长1245nm,半带宽13nm，峰值透过率90%
  - 中心波长1310nm,半带宽25nm,峰值透过率80%
- **label的对应关系**
  - 1: PA
  - 2: PET
  - 3: PMMA
  - 4: PP
  - 5: PS
  - 6: PVC
  - 7: PC
  - 8: PE
